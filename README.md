# ABOUT THE TASK

 PROJECT OVERVIEW
 In this project i have created a responsive landing page for a laundry serviecs website. This page includes a navigation bar,a hero section with txt and img , and call-to-action button.the main objective was to make the media query fo the desktop , tablet and mobile.

 # FEATURES
I HAVE USED FOLLOWING FEATURES:-

Responsive navigation bar
Hero section with service information
Call-to-action button
Flexbox layout
Tablet and mobile responsive design
Image and content adjust according to screen size

# TECHNOLOGIES USED

HTML
CSS
FLEX BOX
MEDIA QUERIES

# THE RESPONSIVE DESIGNS ARE

Desktop View
Navigation links are displayed in a row.
Text content and image are shown side by side.
Larger font sizes are used for better visibility.

Table View  (1024px and below)
Font sizes are slightly reduced.
Spacing is adjusted to fit smaller screens.
Content and image remain side by side.

Tablet View (some tablet has the 766px view so a have created two media query for tablet view)(768px and below)
Font sizes are slightly reduced.
Spacing is adjusted to fit smaller screens.
Content and image remain side by side.

Mobile View (600px and below)
Navigation links are hidden to save space.
Only the logo and username are shown in the navbar.
Hero section changes to a vertical layout.
Text appears above the image.
